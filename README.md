# NodeJsCI-T
NodeJs Challenges

## Instruções para Instalação

1. Instale as dependencias

    npm install
    
2. Crie um arquivo de config.js dentro da pasta config com o seguinte formato

  module.exports = {
    user: 'user',
    password: 'password'
  }

3. node app.js

4. Abra o browser no endereço http://localhost:8080

5. Enjoy
