# NodeJsCI-T
NodeJs Challenges

## Instruções para Instalação

1. Instale as dependencias

```ssh
npm install
```
    
2. Crie um arquivo de config.js dentro da pasta config com o seguinte formato

```js
module.exports = {  
    user: 'user', 
    password: 'password'
}
```

3. node app.js

4. Abra o browser no endereço http://localhost:8080/people

5. Enjoy
